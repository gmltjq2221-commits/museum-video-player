# 박물관 영상 플레이어 — 영상별 별도 주소

주소는 아래처럼 분리됩니다.

- `/` : 첫 화면
- `/video1/` : 1번 영상
- `/video2/` : 2번 영상
- `/video3/` : 3번 영상
- `/video4/` : 4번 영상

예: `https://lhs.r-e.kr/video1/`

## 기존 저장소에 적용하는 방법

1. 기존 `index.html`, `style.css` 파일을 새 파일로 교체합니다.
2. `player.js` 파일을 저장소 최상단에 업로드합니다.
3. `video1`, `video2`, `video3`, `video4` 폴더를 통째로 업로드합니다.
4. 기존 `videos` 폴더 및 `video1.mp4`~`video4.mp4` 파일은 그대로 둡니다.

각 영상 페이지는 자동 반복 없이 재생, 일시정지, 처음부터 재생, 스피커 소리 조절만 제공합니다.
